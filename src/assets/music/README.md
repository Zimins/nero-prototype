# 음악 파일 디렉토리

이 디렉토리에 쥬크박스에서 재생할 MP3 파일을 추가하세요.

## 필요한 파일

다음 파일명으로 MP3 파일을 추가해주세요:

1. `song1.mp3` - 픽셀 카페의 오후
2. `song2.mp3` - 도트 세계의 산책
3. `song3.mp3` - 따뜻한 벽난로
4. `song4.mp3` - 오래된 사진첩
5. `song5.mp3` - 책과 함께하는 시간

## 파일 형식

- 형식: MP3
- 권장 비트레이트: 128kbps 이상
- 모든 브라우저와 호환됩니다

## 무료 음악 소스

무료 음악을 다운로드할 수 있는 사이트:
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/music/)
- [Bensound](https://www.bensound.com/)

## 주의사항

저작권이 있는 음악은 사용하지 마세요. 개인 프로젝트나 학습 목적으로만 사용하세요.
